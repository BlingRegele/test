# Reguli ce privesc chat-ul

1. Scrierea pe chat-ul global sau vorbirea in joc este permisa doar in limba romana
2. Free chat-ul sau spamul pe chat-ul global este interzis. Spam se considera scrierea aceluiasi mesaj de 3 sau mai multe ori consecutiv. Este considerat spam si daca scriu acelasi mesaj mai multe persoane, fiecare dintre acestea fiind sanctionate individual.
3. Folosirea cuvintelor licentioase pe chat-ul global este strict interzisa. Sanctiunea poate varia in functie de gravitate.
4. Folosirea Caps Lock-ului intr-un mod abuziv pe chat este interzisa.

#### Mixing-ul <a href="#mixing-ul" id="mixing-ul"></a>

Mixing-ul reprezinta transmiterea de informatii IC ( In Character ) in OOC ( Out of Character ). Nu aveti voie sa utilizati chat-ul din joc pentru informatii OOC.

**Tipuri de mixing:**

* Mixing-ul care afecteaza roleplay-ul ( ex: "mafia x este la productia de y" ).
* Mixing-ul care nu afecteaza roleplay-ul ( ex: "vand x masina la y pret" ).

De asemenea, mixing-ul se poate produce si pe discord, cel care transmite informatia fiind sanctionat cu mixing iar cel care preia informatia fiind sanctionat cu metagaming.
